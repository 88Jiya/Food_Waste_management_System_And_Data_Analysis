# Food Waste Data Analysis Project

## Project Overview

This project analyzes food waste data to identify patterns, high-waste locations, and trends in food wastage.
Using data analysis and visualization techniques, the project helps understand how food waste is distributed and where improvements in waste management can be made.


## Objectives

* Analyze food waste data to identify patterns.
* Identify locations with the highest food waste.
* Visualize food waste trends using graphs.
* Display food waste distribution on a geographic map.
* Generate insights that can help improve food waste management strategies.


## Tools & Technologies

* **Python**
* **Pandas**
* **Matplotlib**
* **Folium (Map Visualization)**
* **SQL**
* **Power BI**

## Project Files

* **fwa.ipynb** – Jupyter Notebook containing the full data analysis
* **fwt2.csv** – Dataset used for the analysis
* **fw.sql** – SQL queries used for data analysis
* **fwsm.bi** – Power BI dashboard file for visualization

## Features of the Project

* Data cleaning and preprocessing
* Food waste trend analysis
* Top waste location analysis
* Data visualization using graphs
* Geographic visualization using map markers
* Dashboard visualization in Power BI



## Key Insights

* Certain locations generate significantly higher food waste.
* Data visualization helps identify patterns in food wastage.
* waste locations helps understand geographic distribution.



## Visualization

The project includes graphs and a geographic map to visualize food waste locations and waste trends.

##  Conclusion

This project demonstrates how data analysis techniques can be used to understand food waste patterns and identify areas with higher waste generation.  
By using Python, SQL, and Power BI, the project provides meaningful insights through data visualization and geographic mapping.  

The results highlight the importance of data-driven decision making for improving food waste management and promoting more sustainable practices.
